# Taklamakan Fault Visualization

Interactive simulation of tectonic plate boundaries.

## Features

- Transform, divergent, and convergent faults
- Triple junction scenarios (Afar, Mendocino, Chile)
- Real-time displacement controls

## Stack

React · TypeScript · Tailwind · Framer Motion · Vite

## Run

```bash
pnpm install
pnpm dev
```
